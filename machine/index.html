המכונה: שלוש שכבות

רוב האנשים מכירים רק את הצ'אט. מאחוריו יש שלוש שכבות - פלאגינים, סקילים ו-MCP - שכל אחד יכול להתקין, גם בלי רקע טכני. ככה בניתי מהן מכונה.

הרעיון בשורה אחת
כל שכבה מוסיפה כוח משלה: צוות, ידע, וגישה חיה למערכות שלכם.

שלוש השכבות

  1. פלאגין - צוות שלם בהתקנה אחת.
  2. סקיל - ידע שנטען בדיוק כשצריך.
  3. MCP - גישה חיה למערכות שלכם.
מ-2026 סקילים הם תקן פתוח: אותו קובץ רץ גם ב-Copilot, ב-Cursor וגם ב-Gemini CLI - כ-40 כלים כבר תומכים.
